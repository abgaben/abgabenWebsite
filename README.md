# Abgaben Website In diesem Repository pflegen wir die [Abgaben
Website](https://www.abgaben.github.io/abgabenWebsite/) auf der der
Name Programm ist. Des weiteren werden hier auch LATeX Snippets
veröffentlicht, die wir als für die Allgemeinheit interessant
befinden. Alle hier veröffentlichten Dokumente unterliegen, solange
wir Urheber sind, der
[MIT-Linzenz](https://raw.githubusercontent.com/abgaben/abgabenWebsite/master/LICENSE). Im
Sinne dieser Lizenz sind die, auf der Seite enthaltenen Dokumente, als
Software zu betrachten.
